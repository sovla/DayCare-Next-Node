## Lib

bootstrap - 디자인 프레임워크
storybook - 컴포넌트 시각화 툴

-   a11y - 웹 표준 접근성 테스트
-   interactions - 상호작용 및 테스트의 시각적 디버깅
-   actions - 이벤트 핸들러 캐치
-   controls - 동적으로 Props 변화를 위한 툴
-   viewport - 반응형 요소를 위한 툴

styled-components - CSS in JS
ES Lint - 공통적인 코드스타일 적용
